### 这是PredefinOtto-3D (Tridimensionale)附加模型包
这是一个mincraft材质包

用于SamXU1322的X8-Default材质包的**附加模型包**  

计划为所有方块添加额外的模型

进度  3%  [=>------------------] 80/1794

虽然材质本身为8X8的轻量材质，但由于增加了较多的表面，需要较强力的电脑。
(例如工作台由153个小方块构成）

附属有PredefinOtto-max材质包，用于给那些想要寻找刺激的人(RTX3090 3fps)
